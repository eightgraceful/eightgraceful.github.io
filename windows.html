Windows Login Instructions

Login using Windows

  1. Find the Notepad++ application.
  2. Choose "Open file".
  3. Go to the Windows -> System32 -> drivers -> etc page.
  4. Select the "hosts" file and click "open".
  5. Paste the NSPS IP at the very bottom:
  6. 141.94.199.124 www.growtopia1.com
141.94.199.124 www.growtopia2.com
141.94.199.124 growtopia1.com
141.94.199.124 growtopia2.com
  7. Press the CTRL + S key on your keyboard.
  8. Click "Yes".
  9. Press the CTRL + S key again on your keyboard.
  10. Open Growtopia.
